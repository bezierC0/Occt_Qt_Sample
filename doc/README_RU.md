## Русский (Russian)

### Введение
**Occt_Qt_Sample** — это шаблонный проект, демонстрирующий интеграцию **OpenCASCADE Technology (OCCT)** с **Qt5**. Он служит базовой кодовой базой для разработки приложений CAD/CAM/CAE. Этот проект предоставляет чистую структуру, базовую 3D-визуализацию, а также возможности импорта/экспорта файлов CAD, что делает его идеальной отправной точкой для ваших собственных проектов.

### Возможности
*   **Импорт файлов CAD**: Поддержка загрузки файлов `.step`, `.stp`, `.iges`, `.igs` и `.brep`.
*   **Экспорт файлов CAD**: Поддержка экспорта в форматы `.step`, `.iges` и `.brep`.
*   **3D-визуализация**: Интерактивный 3D-просмотрщик на основе OpenCASCADE V3d_View и AIS (Application Interactive Services).
*   **Структура сборки**: Отображение иерархии импортированной CAD-сборки в виде дерева.
*   **Система сборки CMake**: Легко настраиваемая конфигурация сборки CMake.
*   **Чистая архитектура**: Разделение основной логики CAD (`core_api`) и пользовательского интерфейса (`ui`).

### Требования
*   **Qt 5**: Рекомендуется версия 5.12 или выше.
*   **OpenCASCADE Technology**: Рекомендуется версия 7.6.0 или выше.
*   **CMake**: Версия 3.19 или выше.
*   **Компилятор**: Компилятор, совместимый с C++17 (MSVC, GCC, Clang).

### Инструкции по сборке

1.  **Клонирование репозитория**:
    ```bash
    git clone https://github.com/your-username/Occt_Qt_Sample.git
    cd Occt_Qt_Sample
    ```

2.  **Настройка с помощью CMake**:
    Вам необходимо указать путь к установке OpenCASCADE.
    ```bash
    mkdir build
    cd build
    cmake .. -DOpenCASCADE_DIR="C:/OpenCASCADE-7.6.0-vc14-64/opencascade-7.6.0"
    ```
    *Примечание: `OpenCASCADE_DIR` должен указывать на каталог, содержащий `OpenCASCADEConfig.cmake`.*

3.  **Сборка**:
    ```bash
    cmake --build . --config Release
    ```

4.  **Запуск**:
    Исполняемый файл (и необходимые DLL в Windows) будет находиться в папке сборки (например, `build/Release`).
